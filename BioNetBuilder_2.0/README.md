* __Plugin Name__
  * BioNetBuilder
* __Version__
  * 2.0
* __Description__
  * Accesses BIND,BioGrid,DIP,HPRD,KEGG,IntAct,MINT,MPPI, and Prolinks as well as interolog networks derived from these sources for all species represented in NCBI HomoloGene.
* __Category__
  * Network generation
* __Release Date__
  * 2008-06-18
* __Minimum Cytoscape Version__
  * 2.6
