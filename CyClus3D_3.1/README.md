* __Plugin Name__
  * CyClus3D
* __Version__
  * 3.1
* __Description__
  * CyClus3D is a plugin for identifying functional modules in integrated networks composed of multiple interaction types. CyClus3D operates by clustering user-defined 3-node network motifs.
* __Category__
  * Network generation
