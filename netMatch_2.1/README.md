* __Plugin Name__
  * netMatch
* __Version__
  * 2.1
* __Description__
  * NetMatch is a Cytoscape plugin that finds user defined network motifs in any Cytoscape network. Node and edge attributes of any type and paths of unknown length can be specified in the search.
* __Category__
  * Network generation
